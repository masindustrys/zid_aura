# Zid Aura Theme

A comprehensive, modern e-commerce theme for the Zid platform, designed to work across all business niches.

## Features

### 🎨 Modern Design
- Clean, professional design suitable for any industry
- Responsive layout that works on all devices
- Customizable color schemes and typography
- Dark mode support

### 🌐 Multilingual Support
- Full RTL (Right-to-Left) support for Arabic
- LTR support for English and other languages
- Easy language switching

### 🛒 E-commerce Features
- Product showcase with multiple layout options
- Advanced filtering and search
- Quick view functionality
- Wishlist and compare features
- AJAX-powered shopping cart
- Streamlined checkout process

### 📱 Mobile Optimized
- Mobile-first responsive design
- Touch-friendly interface
- Fast loading times
- Progressive Web App ready

### 🔧 Customizable
- Multiple color schemes
- Typography options
- Layout configurations
- Industry-specific templates

### 📈 SEO Optimized
- Clean semantic HTML
- Structured data markup
- Optimized meta tags
- Fast loading speeds

## Installation

1. Upload the theme files to your Zid store
2. Activate the theme in your admin panel
3. Customize colors, fonts, and layout to match your brand
4. Add your content and products

## Customization

### Colors
- Primary color: Brand main color
- Secondary color: Supporting color
- Accent color: Call-to-action color
- Background: Main background color
- Text: Primary text color

### Typography
- Primary font: Main content font (default: Inter)
- Secondary font: Headings font (default: Playfair Display)
- Configurable font sizes

### Layout
- Container width: Maximum content width
- Sidebar width: Sidebar dimensions
- Header height: Top navigation height
