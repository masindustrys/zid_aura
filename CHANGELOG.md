### Files Structure
- `theme.json` - Theme configuration and metadata
- `templates/` - Liquid template files
  - `layouts/theme.liquid` - Main layout template
  - `pages/` - Page templates (home, product, collection, cart, search)
  - `partials/` - Reusable components
- `assets/` - CSS, JavaScript, and image files
  - `css/` - Stylesheets (main, responsive, RTL, dark mode)
  - `js/` - JavaScript functionality
  - `images/` - Theme images and icons
- `locales/` - Translation files
- `config/` - Theme settings schema
- `sections/` - Theme sections for customization
